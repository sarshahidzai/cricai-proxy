
# CRICAI Proxy Server v2
Full cricket proxy for Cricbuzz + ESPN + Yahoo.

## Endpoints
- `/cricbuzz` → Fetches Cricbuzz HTML
- `/espn` → Fetches ESPN cricket JSON
- `/yahoo` → Fetches Yahoo cricket page

## Deploy on Render
1. Push this repo to GitHub.
2. Create new Web Service → Select repository.
3. Render reads `render.yaml` automatically.
4. Click Deploy.
